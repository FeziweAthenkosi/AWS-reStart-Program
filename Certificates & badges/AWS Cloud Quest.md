# AWS Cloud Quest: Cloud Practitioner

## 📖 Overview
This repository documents my journey through **AWS Cloud Quest: Cloud Practitioner**, an interactive, role-based learning experience provided by AWS. Throughout this quest, I tackled real-world business scenarios by designing, deploying, and managing cloud solutions, moving beyond theory into practical application.

## 💡 What I Learned
The Cloud Quest curriculum focused on the foundational pillars of the AWS Cloud. Key areas of competency include:

*   **Cloud Concepts:** Understanding the value proposition of the cloud, including economies of scale, agility, and the Shared Responsibility Model.
*   **Security & Compliance:** Implementing best practices using AWS IAM, security groups, and understanding the AWS compliance framework.
*   **Cloud Technology:** Architecting solutions using core compute, storage, and database services.
*   **Billing & Pricing:** Learning how to manage cloud costs, utilize AWS budgets, and understand different pricing models.
*   **Operational Excellence:** Deploying and monitoring resources to ensure high availability and fault tolerance.

## 🛠️ AWS Services Used
*   **Compute:** Amazon EC2, AWS Lambda
*   **Storage:** Amazon S3, Amazon EBS
*   **Database:** Amazon RDS, Amazon DynamoDB
*   **Networking:** Amazon VPC, Route 53
*   **Security/Identity:** AWS IAM, AWS CloudTrail
*   **Management:** AWS CloudFormation, AWS Trusted Advisor

## 🚀 Skills Enhanced
*   **Problem-Solving:** Applying AWS services to solve specific "client" requirements within the game scenarios.
*   **Architectural Thinking:** Learning to choose the "Right Tool for the Job" (e.g., when to use S3 vs. EBS).
*   **Infrastructure Fundamentals:** Gaining a deep understanding of how networking, compute, and storage work together in a cloud environment.
*   **AWS Best Practices:** Learning to build secure, cost-effective, and scalable architectures from the ground up.

## 🎯 Achievement
Successfully completed all modules and challenges within the Cloud Quest, demonstrating proficiency aligned with the **AWS Certified Cloud Practitioner**
